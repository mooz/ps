Hooks

You can set arbitrary functions to hook points.

Sample

ps.addHook("beforeDisplayPage", function (page) {
    $(page).css("opacity", 1.0);
});

ps.addHook("afterTransitPage", function (page) {
    $(page).css("opacity", 0.3);
});
